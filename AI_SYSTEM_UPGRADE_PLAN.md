# AI 交易系统升级方案 (2026-01-16)

**背景**: 经过与 AI 交易员的深度辩论，确认原有的“单月线回调”选股逻辑存在滞后性与幸存者偏差。需引入“周线战术沙盘”与“量化因子动态调整”来提升策略的鲁棒性。

## 1. 核心理念升级
- **战略地图 (月线)**: 负责定性。筛选主升趋势、确认首次回调、锁定关键支撑位。
- **战术沙盘 (周线)**: 负责验证。确认回调结构 (ABC/楔形)、验证动量背离、寻找入场时机。
- **量化校准**: 引入 ATR (波动率) 动态止损与 OBV (能量潮) 辅助判断。

## 2. 选股逻辑优化 (`stock_selector.py`)

### 新增周线结构过滤器 (`check_weekly_structure`)
- **ABC 结构识别**: 检测周线是否存在“下跌-反弹-下跌”的三浪结构，且 C 浪低点不大幅击穿 A 浪低点 (防止 A 字杀)。
- **动量底背离**: 股价创新低但周线 RSI/MACD 未创新低。
- **均线共振**: 检测月线 MA20 与周线 MA60 的偏离度，优先选共振标的。

### 优化月线初选逻辑
- **幅度动态化**: 废弃 `<20%` 固定阈值，改为 `Drawdown < 2.0 * ATR(Month)`。
- **主升确认**: 要求 `Close > Monthly_MA10 * 1.05` 且 `Monthly_MACD > 0`。

## 3. AI 分析 Prompt 升级 (`prompt_builder.py`)

### 注入双周期视角
- **Mandatory Section**: 强制 AI 分析周线回调结构 (ABC/楔形/单边)。
- **Factor Injection**: 注入 ATR、OBV、RSI 背离状态等量化因子。
- **Risk Control**: 增加“周线破位即止损”的明确指令。

### 评分体系调整
- **结构分 (30%)**: 周线形态是否完整。
- **趋势分 (30%)**: 月线主升是否完好。
- **量能分 (20%)**: 回调是否缩量，OBV 是否背离。
- **其他 (20%)**: 基本面、题材、大盘环境。

## 4. 执行计划
1.  **Phase 1**: 在 `stock_selector.py` 中实现 `_check_weekly_structure` 与 `_calculate_atr`。
2.  **Phase 2**: 修改 `prompt_builder.py`，在 System Prompt 中加入“周线战术”指令，并注入计算好的量化因子。
3.  **Phase 3**: 运行 `test_selection.py` 验证新逻辑选出的标的质量。
